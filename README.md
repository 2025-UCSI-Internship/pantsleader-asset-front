# 2025 UCSI Malaysia Internship - Frontend


---

## 🚀 주요 기능
- 로그인 / 회원가입
- 자산 목록 조회 및 검색
- 자산 상세 정보 확인
- 자산 등록 / 수정
- 자산 이동(대출, 반납) 기록 관리

---

## 🛠 기술 스택
- **Framework**: [SvelteKit](https://kit.svelte.dev/)
- **Language**: JavaScript (ES6+)
- **Styling**: Tailwind CSS
- **Build Tool**: Vite
- **Version Control**: Git / GitHub
